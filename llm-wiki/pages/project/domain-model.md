# 도메인 모델

canonical 용어 정의는 [CONTEXT.md](../../../CONTEXT.md)가 단일 진실이다(이 페이지는 정의를 재작성하지 않고 **관계와 흐름**을 보탠다).

## 핵심 개념과 관계

- **종목 마스터** — 검색·등록 대상 전체 상장종목 목록(코드·종목명·시장·유니버스 포함 여부). 출처는 DART 기업개황.
- **관심종목** — 사용자가 종목 마스터에서 골라 등록한 종목. 브라우저 로컬에만 저장(→ [무로그인 결정](decisions.md)).
- **공시** — 상장사가 공개하는 법정 정보. 발행 시각·공시유형·원문을 가진다.
- **공시 요약** — 하나의 공시를 (1)구조화 발췌 + (2)AI 3줄로 가공한 결과. **사용자 무관하게 공시 단위로 생성·공유**된다.
- **주요 종목 유니버스** — 미리 배치 요약해 두는 종목 집합. KOSPI200 고정 리스트로 정의(사용자 데이터 무관).

## 흐름 (한 사용자 관점)

```
종목 마스터 ──검색·등록──▶ 관심종목(로컬)
                              │
                     관심종목의 공시 조회(기간 필터)
                              │
        ┌─────────────────────┴─────────────────────┐
   유니버스 종목 공시                          유니버스 밖 종목 공시
   = 배치로 미리 요약                          = 조회 시 on-demand 요약
        └─────────────────────┬─────────────────────┘
                       공시 요약(공시 단위 캐시)
                              │
                  피드 / 종목별 보기로 표시 (NEW 뱃지)
```

생성 방식의 근거는 [하이브리드 요약 생성 결정](decisions.md), 코드상 타입은 `Stock`·`Disclosure`([lib/types.ts](../../../lib/types.ts))에 대응한다.
