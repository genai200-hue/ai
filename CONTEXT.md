# 관심종목 공시 요약 (Watchlist Disclosure Summary)

개인투자자가 자신의 관심종목에 대해 새로 올라온 공시를 빠르게 훑어볼 수 있도록, 공시 원문을 요약해 보여주는 웹 애플리케이션의 도메인.

## Language

**관심종목 (Watchlist Item)**:
사용자가 추적하려고 등록해 둔 상장 종목. 하나의 종목은 종목코드로 식별된다.
_Avoid_: 즐겨찾기, 북마크, 포트폴리오(보유종목과 구분됨)

**공시 (Disclosure)**:
상장사가 법령에 따라 공개하는 경영·재무·지분 관련 정보. 하나의 공시는 발행 시각, 공시유형, 원문을 가진다.
_Avoid_: 뉴스, 리포트, IR자료

**공시 요약 (Disclosure Summary)**:
하나의 공시를 사용자가 빠르게 이해하도록 가공한 결과물. 두 층위로 구성된다 —
(1) **구조화 발췌**: 공시 원문에서 뽑은 정형 항목(공시유형·금액·일자·상대방 등),
(2) **AI 요약**: LLM이 원문을 읽고 생성한 핵심 자연어 3줄.
요약은 사용자와 무관하게 **공시 단위**로 만들어져 모든 사용자가 공유·재사용한다.
_Avoid_: 초록, 브리핑

**주요 종목 유니버스 (Core Universe)**:
공시가 올라오는 즉시 미리(배치) 요약해 두는 대상 종목의 집합. 현재는 **KOSPI200 고정 리스트**로 정의하며(정기 리밸런싱 시 수동 갱신), 사용자의 관심종목과는 무관하게 결정된다. 유니버스 밖 종목의 공시는 조회 시점에 요약한다(on-demand).
_Avoid_: 인기 종목(사용자 데이터 기반으로 오해될 수 있어 지양)

**종목 마스터 (Stock Master)**:
검색·등록 대상이 되는 전체 상장종목 목록. DART 기업개황(corpCode)을 출처로 하며, 종목코드·종목명·고유번호를 담는다.
